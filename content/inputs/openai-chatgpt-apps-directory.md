The Apps SDK points toward ChatGPT as a distribution surface rather than only a model interface. Apps can appear inside conversations, carry interactive UI, and reach ChatGPT's user base without asking the user to leave the assistant.

For the hybrid social-infrastructure prediction, the useful signal is not that OpenAI wants an app store in the old sense. It is that OpenAI is giving outside actors a way to become addressable participants inside the user's primary AI surface.
